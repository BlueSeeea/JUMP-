# 跳跳方块 - 轻量级跳跃游戏

一款类似《跳一跳》的轻量级网页小游戏，保留"蓄力跳跃+平台闯关"的经典玩法，同时通过轻量化设计、趣味细节和社交裂变降低实现难度，增强对年轻人的吸引力。

## 🎮 游戏特色

- **30秒上手，1-3分钟一局**：简单直观的操作方式
- **完美着陆系统**：精准落在平台中心获得额外分数和连击
- **多样化平台**：普通、弹簧、加速、分数平台增加游戏趣味
- **皮肤收集系统**：解锁可爱角色皮肤，满足收集欲
- **实时排行榜**：与好友比拼高分，激发竞技乐趣
- **社交分享**：一键分享战绩，展示你的跳跃技巧

## 🎯 核心玩法

- **操作方式**：长按鼠标或触屏蓄力，松开后角色向前跳跃
- **蓄力机制**：蓄力时间越长，跳跃距离越远
- **得分规则**：
  - 成功跳上平台：+10分
  - 完美着陆（平台中心）：+5分
  - 连击系统：连续完美着陆触发得分翻倍
- **游戏目标**：尽可能获得高分，挑战排行榜

## 🎨 视觉效果

- **极简卡通风**：几何图形+低饱和马卡龙色
- **渐变背景**：浅蓝→浅紫的柔和渐变
- **动态特效**：
  - 蓄力时显示力度条（蓝→黄→红渐变）
  - 跳跃轨迹虚线
  - 完美着陆星光特效
  - 角色跳跃旋转动画

## 🛠️ 技术实现

- **前端技术**：HTML5 + CSS3 + 原生JavaScript
- **响应式设计**：完美适配移动端和桌面端
- **本地存储**：localStorage保存游戏进度和排行榜
- **零依赖**：无需后端服务，纯前端实现

## 🚀 快速部署

### 部署到Netlify

1. 访问 [Netlify](https://netlify.com)
2. 拖拽项目文件夹到Netlify部署区域
3. 自动部署完成，获得在线游戏链接

### 本地运行

```bash
# 克隆或下载项目
cd New_game

# 直接在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
# 或
npx serve .
```

## 📱 游戏界面

- **首页**：游戏入口、模式选择、皮肤切换、历史最高分
- **游戏界面**：角色、平台、力度条、实时分数、连击提示
- **结束界面**：本局得分、最佳记录对比、分享按钮、再来一局
- **排行榜**：实时排名、昵称设置、好友比拼
- **皮肤商店**：皮肤预览、解锁条件、收集系统

## 🎵 音效系统

- **跳跃音效**：每次跳跃的轻快音效
- **完美着陆**：成功着陆的清脆提示音
- **游戏结束**：失败时的柔和音效
- **震动反馈**：移动端支持震动API

## 📊 数据存储

游戏使用localStorage本地存储：
- 玩家游戏数据（最高分、游戏次数）
- 排行榜数据（前50名玩家）
- 皮肤解锁状态
- 玩家昵称设置

## 🔧 自定义配置

可以通过修改 `script.js` 中的参数来自定义游戏体验：

```javascript
// 游戏难度参数
gameSpeed: 1,              // 游戏速度
platformSpeed: 0.5,        // 平台移动速度
perfectDistance: 10,       // 完美着陆判定范围

// 皮肤解锁条件
skins: {
    cat: { requirement: 500 },        // 500分解锁
    astronaut: { requirement: 1000 },  // 1000分解锁
    rainbow: { requirement: 2000 }     // 2000分解锁
}
```
